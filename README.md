# Galkin
# Movie Library (Личная кинотека)

**Автор:** [Галкин Влад]
**Описание:** Приложение для ведения списка просмотренных фильмов или личной коллекции.

## Инструкция по использованию
1. Введите данные фильма: название, выберите жанр, год и вашу оценку (0-10).
2. Нажмите "Добавить фильм" для сохранения. Данные автоматически запишутся в `movies.json`.
3. Для поиска используйте блок фильтрации: можно выбрать конкретный жанр или ввести год.

## Примеры тестов
- **Корректность ввода:** Если ввести рейтинг "11" или "текст", программа выдаст предупреждение.
- **Фильтрация:** При выборе жанра "Драма" в таблице скроются все остальные жанры.
- **Сохранение:** После перезагрузки приложения все добавленные фильмы останутся в списке.
